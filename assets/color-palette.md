# Palette de Couleurs - Ricash Developer Portal

## Couleurs Principales

| Couleur | R | G | B | Hex | Rôle |
|---------|---|---|---|-----|------|
| Bleu pétrole foncé | 41 | 72 | 90 | #29485A | Titres, header |
| Bleu-vert | 44 | 131 | 135 | #2C8387 | Boutons, CTA |
| Bleu nuit | 22 | 37 | 41 | #162529 | Backgrounds |
| Bleu-gris | 55 | 101 | 113 | #376571 | Sections, cartes |
| Or/Bronze | 174 | 132 | 85 | #AE8455 | Highlights, icônes |
| Noir | 0 | 0 | 0 | #000000 | Texte |

## Utilisation

- **Bleu pétrole foncé (#29485A)**: Utilisé pour les titres principaux, headers, navigation
- **Bleu-vert (#2C8387)**: Utilisé pour les boutons primaires, CTA, liens actifs
- **Bleu nuit (#162529)**: Utilisé pour les backgrounds en mode sombre
- **Bleu-gris (#376571)**: Utilisé pour les sections, cartes, éléments secondaires
- **Or/Bronze (#AE8455)**: Utilisé pour les highlights, icônes importantes, accents
- **Noir (#000000)**: Utilisé pour le texte principal en mode clair
